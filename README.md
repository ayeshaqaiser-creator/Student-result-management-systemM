# Student Result Management System

## Description
This is a simple C++ console-based project developed to manage student result records. It allows users to add student information, calculate results, search records by roll number, display all records, and save data using file handling.

## Features
- Add student record
- Search student by roll number
- Display all student records
- Calculate total, percentage, grade, and status
- Save and load records from a file

## Technologies Used
- C++
- Arrays
- Functions
- File Handling

## How to Run
1. Compile the program.
2. Run the executable file.
3. Select an option from the menu and follow the instructions.
